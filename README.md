# Dominykas Portfolio 🚀

Welcome to my personal portfolio project! 🌐

## Overview 📝

Using my knowledge in CSS and HTML, I've crafted this space to showcase my skills, projects, and journey as a Frontend Developer. The portfolio is a dynamic canvas where creativity meets functionality, and each project is a chapter in my coding adventure.

## Features ✨

- **Home Section:** Introduces myself and my role as a Frontend Developer, offering a glimpse into my coding philosophy.
  
- **Projects Section:** A showcase of my work, each project representing a unique journey of problem-solving, design, and innovation.

- **Navigation:** Seamless navigation facilitated by a fixed header, ensuring a smooth user experience.

- **Contact Links:** Easily connect with me through LinkedIn, GitHub, and Instagram via the footer links.
